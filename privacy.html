Privacy Policy

AI Markdown Extractor — Last updated: October 19, 2025

Summary:

AI Markdown Extractor does not collect, store, or transmit any personal or sensitive user data. All features and processing occur completely client-side within your browser.

1. What this extension does

AI Markdown Extractor allows you to extract and convert website content or selected text into Markdown format. All conversions and processing happen locally inside your browser — no data ever leaves your device or is transmitted to external servers.

2. No data collection

This extension does not collect any form of personal or sensitive data. Specifically:

  • No personal identifying information (name, email, phone number, address, etc.) is collected.
  • No browsing history, URLs, website content, or screenshots are transmitted or stored.
  • No analytics, telemetry, or tracking scripts are included.
  • No third-party data sharing or advertisement tracking is performed.

3. Client-side functionality

All functionalities of this extension operate entirely within the browser. The extension does not make any external network requests or communicate with any external API, database, or cloud service. Any temporary data generated during Markdown conversion exists only in the browser's memory and is discarded immediately after use.

4. Permissions

This extension only uses the minimum required permissions to function properly, such as:

  • activeTab — to access content from the currently active tab for Markdown extraction.
  • contextMenus — to allow quick access to Markdown conversion from the right-click menu.

These permissions are used solely for the extension’s core functionality and not for any kind of data tracking or collection.

5. Data security

Because the extension does not collect or transmit data, there is no user information stored anywhere. The extension performs all actions locally, ensuring that your data remains completely private and secure. You can review the full source code to verify this behavior if the project is open-sourced.

6. Changes to this privacy policy

If any updates are made to this privacy policy (for example, in the event of new features that require data handling), the revised version will be published at this same URL with an updated “Last updated” date.
